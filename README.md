# Klayi Website

Static marketing site for `klayi.ca`.

## Local Development

This site has no build step. Serve the repository root as static files:

```bash
python3 -m http.server 4173
```

Open `http://localhost:4173`.

## Files

- `index.html`: page markup and SEO metadata
- `styles.css`: token-based styling and layout
- `assets/`: logos, favicon assets, and product imagery
- `robots.txt`: crawler directives
- `sitemap.xml`: sitemap for `https://klayi.ca/`
