macOS utility for custom keyboards

Klayi keeps your active QMK layer visible.

Klayi connects to supported keyboards over USB and shows the current layer and active modifiers directly in the macOS status bar. It is built for people who switch layers often and want immediate visual confirmation without breaking flow.

Minimal setup

Flash once, then Klayi can read live layer state.

Setup is intentionally small. To use Klayi with a supported keyboard, flash that keyboard with the Klayi-compatible fork of its official firmware, connect it over USB, and Klayi will surface the current QMK layer and active modifiers in the macOS status bar.

Step 1

Use the firmware fork for your keyboard

Each supported board has a dedicated firmware fork to expose the layer state Klayi needs.

Svalboard firmware fork on GitHub Moonlander firmware fork on GitHub
Step 2

Flash your keyboard

Once the compatible firmware is flashed, Klayi can read live state over USB without additional runtime setup.

This is a one-time device setup, not a recurring workflow.
